About the Project

TextLens is an AI-driven decision support tool designed to bridge the gap between massive online data and human understanding. Using a FastAPI backend and a fine-tuned DistilBERT transformer model, this system transforms unstructured public chatter into actionable emotional intelligence at scale.

  • Model: Sentiment Analysis Neural Network (BERT)
  • Backend: FastAPI & Python
  • Frontend: Modern CSS3 & Vanilla JavaScript
